# Notion Club Countdown & Announcement Page

A modern and responsive event announcement webpage designed for an upcoming Notion Club workshop. The project features a visually appealing UI, live countdown timer, speaker showcase, interactive FAQ section, and smooth animations using HTML, CSS, and JavaScript.

## Features

### Hero Section

* Event Title
* Tagline
* Event Poster / Banner
* Register Button
* Animated Hero Content

### Live Countdown Timer

* Real-time countdown to the event date
* Separate countdown cards for Days, Hours, Minutes, and Seconds
* Automatic updates every second

### Announcement Section

* Event Overview
* Rules and Instructions
* Venue Information
* Event Date
* Event Timeline

### Speakers Section

* Speaker Cards
* Speaker Images
* Roles and Designations
* Hover Effects and Animations

### FAQ Section

* Expandable and Collapsible Questions
* Interactive User Experience
* Smooth Transitions

### Responsive Design

* Mobile-Friendly Layout
* Flexible Speaker Cards
* Adaptive Sections
* Responsive Countdown Timer

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
